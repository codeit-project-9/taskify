> Taskify는 팀원들과 일정을 공유하고 관리하기 위해 만들어진 서비스입니다. 자유롭게 대시보드를 생성하고, 대시보드 안에서 여러 칼럼들을 두어 할 일을 관리할 수 있습니다. 또한 이메일로 팀원을 자신의 대시보드에 초대해 할 일을 공유할 수 있습니다.

<br>
